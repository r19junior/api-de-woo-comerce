# Fase 2: Herramientas y Primeros Pasos

El enfoque aquí es configurar las herramientas de testing de APIs y realizar las primeras conexiones reales.

## Objetivos
- [ ] **Postman/Insomnia:** Instalar una de estas herramientas y configurar un entorno con variables `url`, `consumer_key` y `consumer_secret`.
- [ ] **Permalinks:** Verificar que `/%postname%/` está activo para que la API responda en `/wp-json/wc/v3/`.
- [ ] **Lectura de Datos:** Realizar un `GET` a `/products` y observar la respuesta.

## Recursos Útiles
- [Documentación de Postman](https://learning.postman.com/)
- [WooCommerce API Reference](https://woocommerce.github.io/woocommerce-rest-api-docs/)

## Reto Práctico
Usa Postman para listar solo los productos que tengan un precio superior a 50 USD usando parámetros en la URL.
