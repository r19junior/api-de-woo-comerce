# Fase 1: Fundamentos

En esta fase el objetivo es entender el ecosistema de WordPress y cómo se comunica con el exterior a través de HTTP y JSON.

## Objetivos
- [ ] **WordPress Core:** Investigar qué es un `Custom Post Type` (CPT) y cómo la `WP REST API` permite consultarlos.
- [ ] **JSON & HTTP:** Practicar la estructura de un objeto JSON y entender los verbos HTTP (`GET`, `POST`, `PUT`, `DELETE`).
- [ ] **Autenticación:** Generar las credenciales en WooCommerce.

## Recursos Útiles
- [Estructura de la API REST de WordPress](https://developer.wordpress.org/rest-api/)
- [Introducción a JSON](https://www.json.org/json-es.html)

## Reto Práctico
Dibuja un diagrama sencillo de cómo viaja un pedido desde que el cliente hace clic en "Comprar" hasta que llega a la base de datos de WordPress.
