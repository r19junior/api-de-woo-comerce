# Fase 3: CRUD y Lógica de Negocio

Pasamos de la lectura a la acción: crear, actualizar y borrar datos, además de reaccionar a eventos automáticos.

## Objetivos
- [ ] **Creación y Edición:** Crear un producto de prueba vía `POST` y luego cambiar su stock vía `PUT`.
- [ ] **Paginación:** Aprender a usar los headers `X-WP-Total` y `X-WP-TotalPages`.
- [ ] **Webhooks:** Crear un webhook que se dispare cuando un pedido cambie a "Completado".

## Conceptos Avanzados
- **Batch Update:** Cómo enviar 10 actualizaciones en una sola petición.
- **Stock Management:** Lógica detrás de la reducción de inventario.

## Reto Práctico
Crea un script (o usa Postman) para actualizar el precio de 5 productos al mismo tiempo usando la funcionalidad de batch.
